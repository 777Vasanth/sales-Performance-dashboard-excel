# 📊 Sales Performance Dashboard (Excel)

## 📌 Overview

This project presents an interactive Sales Performance Dashboard built using Microsoft Excel to analyze business performance across multiple dimensions such as revenue, profit, customer segments, and regional sales.

The dashboard enables users to explore key metrics and uncover insights that support data-driven decision-making.

---

## 🎯 Key Metrics

* Total Sales: ₹5.39L
* Total Orders: 400
* Total Profit: ₹997K
* Average Order Value: ₹13.5K

---

## 🔍 Key Insights

* Corporate segment contributes the highest share of revenue
* Profit varies significantly across brands, indicating uneven profitability
* Peak sales observed in February and December
* Maharashtra and West Bengal are the top-performing states

---

## 📊 Features

* Interactive slicers for filtering by State, Payment Mode, Customer Segment, and Category
* Monthly sales trend analysis
* Profit comparison across brands
* Category-wise performance evaluation
* Customer segment analysis

---

## 🛠 Tools & Techniques

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning & Transformation
